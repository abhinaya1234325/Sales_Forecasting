# 📈 Sales Forecasting Dashboard – Power BI

This project visualizes sales trends using Power BI. It includes:

- 💰 Total Sales and Average Monthly Sales KPIs
- 🌍 Country and City filters
- 📊 Line, bar, map, and pie charts
- 🔎 Slicers for interactivity

## 🚀 Tools Used
- Power BI Desktop
- DAX Measures
- Sample Sales Dataset
